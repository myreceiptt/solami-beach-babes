## Usage

### Install dependencies

```bash
yarn
```

### Check outdated dependencies

```bash
yarn outdated
```

### Upgrade dependencies interactively

```bash
yarn upgrade-interactive --latest
```

### Start development server

```bash
yarn dev
```

### Check all the code

```bash
yarn run lint
```

### Create a production build

```bash
yarn build
```

### Preview the production build

```bash
yarn start
```
